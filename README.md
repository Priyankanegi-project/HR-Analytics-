# HR-Analytics-
HR Analytics project using SQL, Python, and Power BI to analyze employee data,sales trend,performance and working insight
## tools used
1.Power Bi
2.Python(Pandas,Numpy)
3.PostgreSql


## Workflow
1.collected raw HR data from kaggle.
2.Cleaned and transformed the data using python.
3.Imported cleaned data into PostgreSQL.
4.Perfored SQL analysis.
5.Connected PostgreSql to Power BI
6.Built an interactive dashboard

## Dashboard
1.Total Employees
2.Average Salary
3.Average Age
4.Average Rate
